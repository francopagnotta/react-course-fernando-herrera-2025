# React Course - Fernando Herrera 2025

This repository contains the code and exercises from the React course by Fernando Herrera (2025 edition).

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```
